Hypertext Markup Language (HTML) is the standard markup language[a] for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript, a programming language.


A web browser (often shortened to browser) is an application for accessing web pages and navigating the Internet. When a user visits a website, the browser requests the web page from a web server and displays it. Browsers can also be used to access web pages hosted on the local filesystem (often called localhost) or network configuration pages.


Browsers are used on a range of devices, including desktops, laptops, tablets, and smartphones. As of 2025, the most used browsers worldwide are Google Chrome (66% market share), Safari (18%), Edge (5%), Firefox (3%), Samsung Internet (2%), and Opera (2%).[1] As of 2023, an estimated 5.4 billion people had used a browser.[2]
